# Lists, Queues and Stacks




## Challenges - Javascript

### Stack and Queue

**Question:** Implement a stack and queue.

**Answer:** The stack and queue data structures already exist in the form of arrays with push, pop and shift operators.

``` javascript
var myStack = [];

myStack.push(1);
myStack.push(2);
myStack.pop();

var myQueue = [];

myQueue.push(1);
myQueue.push(2);
myQueue.shift();

```
